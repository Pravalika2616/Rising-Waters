Model Operations

Translating Weather Data Into Safety Action

The engine operates by loading a pre-trained **Extreme Gradient Boosting (XGBoost)** classifier alongside a pre-fitted **StandardScaler** mapping profile. When input variables are entered, the system standardizes the scale to eliminate numerical skewness and evaluates the data across multi-branched decision boundaries.

This system serves as an interactive decision-support utility for meteorologists, relief workers, and agricultural analysts, replacing heavy physical boundary simulations with rapid, reliable classification boundaries.
